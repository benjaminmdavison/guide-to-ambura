# Encyclopedia Ambura: Readme

This project is an atlas detailing the fictional world of Ambura, which I built while running a D&D campaign with my brother. It is a simple HTML/CSS website without a lot of bells and whistles. It is intended as a reference page, and therefore doesn't need to be overly complicated. There's something to be said for keeping it simple. 

# Statement on AI Use  
Github Copilot was used to assist with specific tasks in the generation of this website. Specifically:
- Text formatting of pasted content (game content was written as markdown files),
- Generation of CSS,
- Repetitive generation of templated HTML pages, e.g., Atlas and Bestiary entries. See extensive marginalia below. 
    - Yes, yes. Repetitive tasks do indicate that something can be improved, and in principle, I absolutely agree with that. However:
    - Why not just use a single HTML page and feed it from a JSON object? This would probably be a 'simpler' approach in a lot of ways, but it also relies on making and maintaining enormous JSON objects, which is honestly a pain in the ass. I'd rather do it this way. That said, this will likely evolve over time.
    - Why not use Jekyll? I tried out Jekyll and I didn't like it. It's very popular, so I think it's probably good for a lot of things, but I just kind of want to do it the old-fashioned way. So that's what's happening. For now. 


No generative AI was used to generate the content or concepts in this game module.
