# Encyclopeda Ambura V2.0

We are building a directory-type structure to organize information. Each page and any subpages is contained in it's own folder, except for main.html.

## Site Structure

main.html links to three different pages:
- Campaign notes (campaign.html)
- Bestiary (bestiary.html)
- Atlas (atlas.html)

Each of these pages contains a list of subpages.  

There is also a page titled dmg.html, which is NOT linked from main.html. 

### Campaign Notes
The Campaign Notes page is intended to house campaign history for reference by the DM and the party. It is one long HTML document with headings for each session of play. 

### Bestiary
The Bestiary has links to the following pages. Each page is it's own HTML document, e.g., "mountains.html":
- Belowground
- Cities
- Forests
- Magical
- Mountains
- Mythical
- Oceans, Lakes, and Waterways
- Plains 
- Swamps

### Atlas
The Atlas has links to the following pages. Each page is it's own HTML document. Some pages have subpages: e.g., under breland.html is another page called brelasshield.html.
- Breland
-- Brelas Shield
-- The Crossroads
-- The Free City of Porphyrium
- Borovia
- St. Alberich
- Bramblewood
- South Seas
- Dragonback Mountains


## Page Structure
Each page has the following items:
- Page title: Encyclopedia Ambura
- Subpage title: e.g., "Bestiary", "Atlas", "Belowground"
- Page style provided by style.css
- Navbar: the navigation bar at the top of each page should include links to main, atlas, and bestiary. 
- Shell content: an empty H1 and an empty paragraph.  
- Footer with content "Copyright 2026 Benjamin Davison"
